# Background Ambiance Audio Files

This directory should contain ambient background sounds for the conversation feature.

## Required Files

Add the following MP3 files (royalty-free ambient sounds):

| File | Description | Suggested Sources |
|------|-------------|-------------------|
| `cafe.mp3` | Coffee shop ambiance - chatter, espresso machines | freesound.org, pixabay.com |
| `office.mp3` | Office sounds - keyboard typing, printers | freesound.org, pixabay.com |
| `restaurant.mp3` | Restaurant ambiance - dishes, conversations | freesound.org, pixabay.com |
| `hospital.mp3` | Hospital/clinic ambiance - quiet halls | freesound.org, pixabay.com |
| `street.mp3` | City street sounds - traffic, pedestrians | freesound.org, pixabay.com |
| `train.mp3` | Train station - announcements, crowds | freesound.org, pixabay.com |
| `airport.mp3` | Airport ambiance - announcements, crowds | freesound.org, pixabay.com |
| `park.mp3` | Park/nature - birds, wind, outdoors | freesound.org, pixabay.com |
| `home.mp3` | Quiet indoor ambiance | freesound.org, pixabay.com |
| `supermarket.mp3` | Supermarket sounds - carts, beeping | freesound.org, pixabay.com |
| `library.mp3` | Very quiet - page turns, whispers | freesound.org, pixabay.com |

## Recommendations

1. **File Size**: Keep files small (under 1MB each) - they loop continuously
2. **Length**: 30-60 seconds is ideal for seamless looping
3. **Format**: MP3 at 128kbps is sufficient
4. **Volume**: Normalize to -20dB for consistent playback
5. **License**: Use royalty-free sounds only

## Sources for Free Ambient Sounds

- [Freesound.org](https://freesound.org) - CC licensed sounds
